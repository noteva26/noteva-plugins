# 友情链接插件

在自定义页面展示友情链接，支持分类、自定义样式和多种布局。

## 功能特性

- ✅ 网格/列表两种布局
- ✅ 支持友链分类
- ✅ 支持 Logo 和描述
- ✅ 响应式设计
- ✅ 暗色模式支持
- ✅ 悬停动画效果

## 使用方法

### 1. 安装插件

将插件文件夹放到 `plugins/` 目录下，重启 Noteva 或调用热重载 API。

### 2. 创建友链页面

在管理后台创建一个自定义页面：
- 标题：友情链接（随意）
- Slug：`friendlinks`（或其他自定义）
- 内容：随意填写（会被插件替换）

### 3. 配置插件

进入 **管理后台 → 插件 → 友情链接 → 设置**：

#### 基本设置
- **目标页面 Slug**：填写刚才创建的页面 slug（如 `friendlinks`）
- **页面标题**：友链页面显示的标题
- **页面描述**：显示在友链列表上方的描述

#### 友链列表
编辑 JSON 格式的友链数据，每个友链包含：
```json
[
  {
    "name": "站点名称",
    "url": "https://example.com",
    "logo": "https://example.com/logo.png",
    "description": "站点描述",
    "category": "分类名称"
  }
]
```

字段说明：
- `name`：必填，站点名称
- `url`：必填，站点链接
- `logo`：可选，Logo 图片 URL
- `description`：可选，站点描述
- `category`：可选，分类名称（不填则归入"未分类"）

#### 样式设置
- **布局方式**：网格布局 / 列表布局
- **每行显示数量**：1-6（仅网格布局有效）
- **显示 Logo**：是否显示站点 Logo
- **显示描述**：是否显示站点描述

### 4. 启用插件

在插件管理页面启用"友情链接"插件。

### 5. 访问页面

访问 `/friendlinks`（或你设置的 slug）即可看到友链页面。

## 示例配置

```json
[
  {
    "name": "Noteva 官网",
    "url": "https://github.com/noteva26/noteva",
    "logo": "https://avatars.githubusercontent.com/u/xxxxx",
    "description": "轻量级博客系统",
    "category": "推荐"
  },
  {
    "name": "示例博客",
    "url": "https://example.com",
    "logo": "",
    "description": "一个很棒的博客",
    "category": "博客"
  }
]
```

## 友链申请

当前版本（v0.1.0）暂不支持在线申请功能，用户可以通过以下方式申请友链：
- 在留言板或其他文章评论
- 通过邮件联系管理员
- 通过社交媒体联系

管理员收到申请后，在插件设置中手动添加即可。
